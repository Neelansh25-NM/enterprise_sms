# enterprise_sms
a Java based Student Management System that not only allows CRUD operations, but also allows dynamic batch filtering for admins based on the specific semester, course and division they are allocated to, viewing the attendance of a student, generating automatic transcripts for a student for the entire term, and also creating assignments.
The admin/faculty can also dynamically add their allocated schedules and lectures for subjects they will teach to students of a SPECIFIC course, semester, and divison.
This project incorporates MySQL (for database), JDBC, jFreeChart, JDK, and lots of code!
